Person eloquent sample on php version 8.4 or later.

## About this package

Person Model

- public name with header 'Mr.' include and initial uppercase
- public age

Model bind check

- name(2 to 30 length)
- age(over 15)

## Update Rules

When updating, execute the following command.

```
herd coverage ./vendor/bin/pest --coverage
art test
```
